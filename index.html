Data literacy is very important these days, especially with the rapidly changing priorities in today's environment. The new generation must be able to interpret data in inform actionable decision making. The goal of this blog is to help teach readers how to interpret data. Let's start with a simple example. I will use the Walkscore API to help users understand how to read walkability and transit scores in the area that they live in. Very excited to start this journey with all of you. Let's begin!
